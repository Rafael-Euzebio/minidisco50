# Changelog

## 2.0.0 (2023-12-06)
### Feature
- Playlists
    - Create Playlists: `./minidisco50 (-a or --add) [songs] [playlist.m3u]`
    - Read Playlists: `./minidisco50 (-r or --read-playlist) [playlist.m3u]`

## 1.1.0 (2023-11-29)
